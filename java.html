O que é Java?

Java é uma linguagem de programação orientada a objetos, projetada para ser multiplataforma, ou seja, o código pode ser executado em diferentes sistemas operacionais usando a Máquina Virtual Java (JVM). Conhecida por sua simplicidade, robustez e segurança, Java é amplamente usada em aplicações empresariais, web e dispositivos móveis (como Android). Ela prioriza portabilidade e desempenho, sendo uma escolha popular para desenvolvimento de grandes sistemas distribuídos e escaláveis.
